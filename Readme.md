# Lab-killer

Веб-приложение для лабораторных вычислений.
Основная часть - это язык программирования, похожий на матлаб, но с физтеховской спецификой.
Документация по языку представленна [здесь](https://github.com/timattt/TypeThree/blob/master/manual.pdf)

## Сущности

* WebClient - приложение на ReactJS. Пользователь пишет программу, потом она улетает на сервер, там воркеры ее выполняют и результат кладут в БД. А клиент каждые несколько секунд пингует **ResultsProvider**, и когда воркеры закончат он получит результат выполнения своего кода.
